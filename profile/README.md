# GitHub Migration Tools

This organization provides tools for the migration to **GitHub Enterprise Cloud (GHEC)**.

## Libraries

### [Octoshift](https://github.com/gh-migration-tools/Octoshift)

This repository extracts the code of the Octoshift library from the gh-gei GitHub Enterprise Importer CLI project and provides it as a NuGet package.

### [Octoshift.Extensions](https://github.com/gh-migration-tools/Octoshift.Extensions)

This project provides a NuGet package with reusable classes based on the [Octoshift](https://github.com/gh-migration-tools/Octoshift) library for custom migration tools.

### [GitHub.Migration](https://github.com/gh-migration-tools/GitHub.Migration)

This project provides a NuGet package with reusable classes for custom migration tools.
